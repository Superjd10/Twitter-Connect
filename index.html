Twiter Connect [PHP]

Es un conjunto de archivos que permiten la autenticación del usuario usando su Twitter y luego añadir sus datos a una base de Datos

Para configurar, solo debes abrir el archivo "config.php" y editar lo que sale en mayúsculas del lado derecho con los respectivos datos.

Este código fue hecho a base de un 95% del código original de "Abraham TwitterOAuth". y con un tutorial de net.tutsplus.com
